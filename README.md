# MRC_Labeling_Tool
#### Input file (write in directory dataset):
```bash
sample file: test.json
format: [{"question":"test","answer":"test"}]
```

#### Run tool:
```bash
- Insert json file in dataset directory
- Run insert_db.py
- Run main.py (and choose "select answer" tag for labeling) 
```

#### Export File
```bash
- Run dump.py
```
